## Getting Help

1. Check the [FAQ](./faq) for common questions
2. Search [GitHub Issues](https://github.com/QuantumNous/new-api/issues)
3. Open a new issue with detailed reproduction steps

## Contributing

We welcome contributions! Please read the contributing guidelines in the GitHub repository before submitting pull requests.

### Translation

Webchannel supports multiple languages. Help translate documentation by submitting PRs to the `new-api-docs` project or using the admin panel (for authorized editors).
